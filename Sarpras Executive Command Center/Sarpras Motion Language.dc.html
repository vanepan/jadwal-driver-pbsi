01
Philosophy

Every motion answers a question.

Nothing here moves because movement is possible. Before any element is allowed to animate, it must answer one question: what operational information is being communicated? If the honest answer is “none,” the element stays still. Stillness is the default state of a healthy operation.

Every sanctioned movement resolves to one of five communications. This is the whole vocabulary — there are no others.

02
Timing & easing

Six rhythms. One easing family.

We do not think in milliseconds; we think in rhythm. Every interaction is assigned one of six named tempos. Durations are the reference range for each — engineering tunes within them, never outside.

cubic-bezier(.2, .7, .2, 1) The standard ease — a gentle out-curve that starts with intent and settles softly. One exception: Urgent uses a firmer decisive curve. No springs, no bounce, ever.

05
Emotional rhythm

The interface has a heartbeat.

Tempo is emotional information. When operations are healthy, motion becomes invisible — the page settles and holds perfectly still, because there is nothing to say. When operations are critical, one restrained pulse keeps a single point alive, and transitions turn crisp and decisive. Same components, different pulse. No flashing, ever.

HEALTHY
Calm · settling · then still
Sangat Baik
Semua berjalan normal.
A single slow entrance, then complete rest. There is no persistent motion — stillness is the reward of a healthy operation.
CRITICAL
Decisive · one quiet pulse
Kritis
Tindakan diperlukan hari ini.
Entrances shorten and firm up; the ring arrives quickly and decisively. One low-amplitude pulse marks the live risk — urgent, never frantic.

08
Reduced motion

Reduced, not removed.

When prefers-reduced-motion is set, the experience stays premium. We do not strip the interface bare — we swap the vocabulary. Positional travel and continuous pulses are removed; a brief, calm opacity settle remains so state changes are still witnessed. The reading order still teaches itself; it simply arrives without movement.

09
Performance & implementation

Cheap by construction. 60fps everywhere.

This runs inside Sarpras Operations on mid-range Android, older iPhones, and low-power laptops. The language is built so that being on-budget is the default, not a later optimisation. Animate only transform and opacity — the two properties the compositor moves without re-painting or re-laying-out the page.

10
Adoption

One language, every module.

Executive Command Center is where this language was written, but it does not own it. Every module inherits the same five communications, six rhythms, and one easing family. Motion is now a shared primitive — a new product is “on-brand” in motion the moment it uses these tokens.
